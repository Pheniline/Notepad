Notepad

A multi-user notes application built with React (Vite)and Node.js/Express.  
Users can sign up, log in, and manage personal notes with support for dark/light themes, note colors, and auto-resizing textareas.  

---

Features

- ✅ User Authentication: Sign up, login, and logout.  
- ✅ Personalized Welcome: Greet users by name after login.  
- ✅ CRUD Notes: Create, read, update, and delete notes.  
- ✅ Note Colors: Customize note background colors.  
- ✅ Auto-Growing Textarea: Expands as you type.  
- ✅ Search Notes: Quickly find notes by keyword.  
- ✅ Theme Toggle: Switch between dark and light modes.  
- ✅ LocalStorage Persistence: Keep user session and theme across page refreshes.  

---

 Tech Stack

- Frontend: React (Vite), CSS, JavaScript  
- Backend: Node.js, Express  
- Data Storage: In-memory (array, can be upgraded to MongoDB or SQL)  
- CORS enabled for frontend-backend communication
